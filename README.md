# Mes premiers projets

Bienvenue !  
Ce dépôt regroupe mes premières expériences et exercices réalisés pendant mon apprentissage du développement web.

## Contenu
- **Projets HTML/CSS** : mes tout premiers exercices et sites simples.
- **Cours HTML** : le premier site que j'ai fait en autodidacte.
- **CV Bootstrap 5.0.2** : un CV réalisé avec Bootstrap pour une première expérience avec ce framework.
- **Mini site avec Bootstrap** : suite de ma première expérience, montrant ce qu’on peut faire avec carousel et cards.
- **Le Globule** : exercice où je devais reproduire un site à l'identique en utilisant uniquement les méthodes classiques, sans flex.
- **Street Life** : suite de l'exercice Le Globule, cette fois avec flexbox et les premiers essais de responsive design.

## Objectif
Montrer ma progression en tant que développeur web débutant et m'entraîner régulièrement sur de nouveaux projets.
